About Project-TMS

Project-TMS

Version 1.0.0

Project-TMS is a modern, feature-rich task management system built with HTML, CSS, and JavaScript. It helps teams track projects, manage tasks, monitor time, and collaborate effectively - all within a beautiful and responsive interface.

Features

  • Project Management
  • Kanban Board
  • Time Tracking
  • Task Details and Notes
  • Analytics
  • Custom Themes
  • Responsive Design
